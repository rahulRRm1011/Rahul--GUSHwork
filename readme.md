Evaluation Criteria Covered:

1. Figma Accuracy
The layout, section order, typography style, spacing, cards, buttons, and product-page structure were implemented based on the provided Figma.

2. Vanilla Code Quality
The project uses only semantic HTML5, modern CSS, and vanilla JavaScript. No frameworks or external UI libraries are used.

3. Sticky Header
The sticky product header appears after the first fold, positions itself above the navigation bar, hides when scrolling back up, and uses smooth transitions.

4. Image Carousel with Zoom
The product gallery includes previous/next controls, clickable thumbnails, active thumbnail state, and desktop hover zoom preview. Zoom is disabled safely on mobile.

5. Responsive Design
The page is responsive across desktop, tablet, and mobile using CSS Grid, Flexbox, and media queries.

6. Accessibility
The page includes meaningful alt text, aria attributes, keyboard-friendly controls, visible focus states, and reduced-motion support.

7. Performance
Images use lazy loading where appropriate, animations use transform/opacity, and scroll behavior is optimized to avoid layout lag.

8. Documentation
Brief comments are included in HTML, CSS, and JavaScript to explain key sections and functionality.
